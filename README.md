# whatsapp-bot
Getting Started
This project require NodeJS v12.

Install
Clone this project

> git clone https://github.com/muhammaddzaky/whatsapp-bot.git
> cd whatsapp-bot
Install the dependencies:

> npm install 
> npm install gify-cli -g
Usage
Run the Whatsapp bot

> npm start
after running it you need to scan the qr
